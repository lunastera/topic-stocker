# 話題ストッカー

気になるページを一時保存しておけるブラウザ拡張機能。

## 機能

- 現在開いているページをワンクリックでストックリストに保存
- ストックしたアイテムをクリックして URL をクリップボードにコピー
- アイテム右端の × ボタンでリストから削除
- ストックはブラウザを閉じても保持される

## 技術スタック

| 項目 | 内容 |
|------|------|
| フレームワーク | [WXT](https://wxt.dev/) 0.20 |
| UI | React 18 + TypeScript |
| i18n | @wxt-dev/i18n |
| ストレージ | wxt/utils/storage |
| マニフェスト | Manifest V3 |
